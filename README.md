# Курсовая по классическому ML

Курсовая работа: попытка предсказать свойства химических соединений против вируса гриппа по табличным молекулярным дескрипторам. Я учусь на магистратуре по данным, химию знаю поверхностно, поэтому в коде больше ML, чем химии.

Итоговый аналитический отчёт лежит в файле `Аналитический_отчёт.docx`. Здесь в репозитории - код и результаты экспериментов.

## Данные

Исходный файл: `Данные_для_курсовои_Классическое_МО.xlsx` (1000 соединений).

Целевые переменные:
- IC50 - ингибирующая концентрация, мМ (меньше обычно лучше)
- CC50 - цитотоксичность, мМ
- SI - индекс селективности, по сути CC50 / IC50 (чем выше, тем интереснее соединение)

Остальные столбцы - числовые дескрипторы молекул (MolWt, MolLogP, TPSA и ещё около 200 штук).

## Что где лежит

`eda.py` - разведочный анализ: распределения целей, пропуски, корреляции, пороги для классификации. Графики и таблицы сохраняются в `outputs/eda/`.

Скрипты моделей лежат в папке `models/`:

регрессия:
- `models/regression_ic50.py`
- `models/regression_cc50.py`
- `models/regression_si.py`

классификация:
- `models/classification_ic50_median.py` - IC50 выше медианы
- `models/classification_cc50_median.py` - CC50 выше медианы
- `models/classification_si_median.py` - SI выше медианы
- `models/classification_si_gt8.py` - SI > 8

`src/` - общие функции загрузки данных и обучения моделей.

`run_all.py` - запускает всё подряд, если не хочется кликать по файлам.

`outputs/` - JSON с метриками после обучения, плюс папка `outputs/eda/` с картинками.

## Как запустить

```bash
pip install -r requirements.txt
python run_all.py
```

Можно и по отдельности, например:

```bash
python eda.py
python -m models.regression_ic50
python -m models.classification_si_gt8
```

## Что я делал по методологии

Сначала убрал константные признаки и заполнил пропуски медианой. Для регрессии взял log1p от целевой переменной - без этого распределения слишком скошены. Разбил данные 80/20, random_state=42.

Сравнивал несколько моделей из sklearn с GridSearchCV на 5 фолдах:
- регрессия: Ridge, ElasticNet, RandomForest, GradientBoosting, SVR
- классификация: LogisticRegression, RandomForest, GradientBoosting, SVC

Метрики регрессии: RMSE, MAE, R2 (на log-шкале, где было преобразование).
Метрики классификации: accuracy, precision, recall, F1, ROC-AUC.

Лучше всего в большинстве задач отработали ансамбли на деревьях. Подробные цифры и мои комментарии - в аналитическом отчёте.
